🐍 Snake & Ladder Game
A terminal-based Snake and Ladder game built with Python. Supports multiple players, snakes, and ladders — all played right in your command line!

🎮 How to Play

Run the script
Enter the number of players
Enter each player's name
Players take turns — press Enter to roll the dice
Land on a 🪜 ladder → move up! Land on a 🐍 snake → slide down!
First player to reach exactly 100 wins 🎉
